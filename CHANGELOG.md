# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-04-01

### Added
- Initial release of KMP Geofence Library
- Cross-platform geofencing support for Android and iOS
- Lambda-based callback system for geofence events
- Permission status checking with detailed error messages
- Add and remove geofences dynamically
- Support for circular geofences
- Automatic detection of user already inside geofence on iOS
- Background location monitoring support
- Comprehensive documentation and usage examples

### Android Features
- Google Play Services Location API integration
- Broadcast receiver for geofence events
- Support for Android 7.0+ (API 24+)
- Background location permission handling for Android 10+

### iOS Features
- CoreLocation framework integration
- Support for iOS 9.0+
- Always authorization handling
- Precise location support for iOS 14+
- Automatic state checking for existing geofences

### Documentation
- Complete README with setup instructions
- Usage examples for common scenarios
- Platform-specific notes and limitations
- API reference documentation
